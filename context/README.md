# Context Folder

This folder contains your personal documents that DeepRecall will index and search.

## Supported Files
- PDF documents (`.pdf`)
- Text files (`.txt`)
- Markdown files (`.md`)

## Usage

**Add your documents:**
   ```bash
   copy "C:\path\to\your\document.pdf" context\
Files are automatically indexed when:

DeepRecall starts

Files are added/modified while running

Privacy:

These files are stored locally only

Never uploaded anywhere

Embeddings cached in cache/embeddings/

Examples
Add your:

Personal notes

Research papers

Technical documentation

Meeting notes

Project documentation

Any knowledge base content

File Limits
No strict size limit

Recommended: < 50MB per PDF for best performance

Large files are chunked automatically
